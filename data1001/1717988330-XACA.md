# RMS

Root Mean Square
$\sqrt{\frac{1}{n}(a_1^2 + a_2^2 + a_3^2 ... + a_n^2)}$
square root of the mean of the values all squared

e.g. values = (1, 2, 3, 4)
RMS of values where n is the length of the list of values:
$\sqrt{\frac{1}{4}(1^2 + 2^2 + 3^2 + 4^2)}$
